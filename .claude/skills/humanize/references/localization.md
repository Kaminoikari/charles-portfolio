# 台灣在地化：用語與標點

中國用語與半形標點是最容易被一眼認出的機器翻譯感。本表為 L0 級：中文對外文字一律套用。

## 判斷原則

1. **語境優先**：指涉中國事物時保留原詞（北京地鐵就是地鐵，不改捷運）；引號內的原話不改。
2. **產業慣用語放行**：台灣科技業日常已通用的詞（迭代、對齊、留存率、上線）不硬改，但抽象黑話（賦能、抓手、閉環）一律處理。
3. **改寫優於替換**：企業黑話不是換一個台灣詞就好，要問「這句話具體想說什麼」，用白話寫出來。

## 標點規範（L0）

1. **全形標點**：中文句子一律用「，。、；：？！」。例外：英文詞組內部、網址、程式碼。
2. **引號**：第一層「」，巢狀『』。**絕不用** `""` `''`。
3. **書名與篇名**：書刊《》，單篇文章〈〉。
4. **刪節號**：「⋯⋯」或「……」（六點），不用 `...`。
5. **破折號**：全形「——」，且受 F2 頻率限制（≤1 次／500 字）。
6. **數字**：統計、金額、日期用半形阿拉伯數字（80 人、4,800 元、42%、2026 年）；約數與慣用語用中文（幾十年、三三兩兩、一個人）。
7. **中英夾雜**：英文與中文之間留一個半形空格（台灣科技媒體慣例）；全文格式一致。
8. **驚嘆號**：全篇 ≤1 個；正式文體 0 個。連續驚嘆號（！！）任何情境都不允許。

## 誤殺防護

以下情況**不改**：

- 專有名詞、品牌名、產品官方名稱（即使含中國用語字樣）
- 引號內的原話、受訪者用語
- 刻意的哏、雙關、模仿語氣
- 指涉中國當地事物的正確用詞
- 使用者已明示要保留的詞
