# 鉤子句式庫（D3 值域）

第一行的唯一任務：給滑動的拇指一個停下來的理由。硬規則：**摺疊線之前（前兩行）必須自帶完整張力**；開頭不自我介紹、不鋪陳、不預告（「今天來聊聊」即死）。

每張卡：公式骨架／樣本（corpus.md 編號）／適配／注意。

### H1 直球宣告
- 公式：〔結論或宣告本身，一句到位〕
- 樣本：「明天放」（34）、「留友看。」（02）、「我自行宣布 第一屆⋯正式開始」（10）
- 適配：T7 幹話、T11 開賽、P7 權威
- 注意：資訊量要真的夠大，否則只是短，不是狠。

### H2 反直覺宣告
- 公式：〔身分該做的事〕的反面（我拒絕了一個病人／流量低才該經營）
- 樣本：「我拒絕了一個糖尿病病人。」（47）、「才 26 個讚⋯換個角度想」（59）
- 適配：P3 專業者、T12 職人內幕
- 注意：後文必須把反直覺圓回來，圓不回來是標題黨。

### H3 否定式喊話
- 公式：不要／別再〔目標受眾正在做的事〕，可三連加強
- 樣本：「『不要，不要，不要再這樣做了』」（53）
- 適配：P2 過來人、P4 經營者、T3 攻略
- 注意：喊完兩行內必須給正確做法，只喊不給＝說教。

### H4 吹牛體
- 公式：〔全民共有的事物〕是〔具體年齡／時間〕的我發明的＋低調補刀
- 樣本：「台灣的超商取貨 是23歲時的我發明的 以防你不知道 我小聲說一下。」（09）
- 適配：P1 素人、T7 幹話
- 注意：牛皮要大到不可能有人信，才安全；P3 專業者慎用（信任稀釋）。

### H5 具體問句
- 公式：你們〔具體場景〕都〔怎麼做／選哪個〕？門檻低到可秒答
- 樣本：「你們覺得最難喝的手搖飲料店是哪間？」（29）、「現在有芒果上市了嗎？」（30）
- 適配：T5 問句徵答
- 注意：心中早有答案還假裝發問，會被做成反諷梗；問就要真問。

### H6 點名集合／徵集
- 公式：各位〔身分標籤〕⋯／歡迎脆友分享〔具體物〕
- 樣本：「各位初級大人 有遇到什麼要找高級大人的事情嗎」（13）、「歡迎脆友分享自己媽媽照片跟年齡」（74）
- 適配：T5、T11、E3 收 UGC
- 注意：身分標籤要正在流行或人人自認符合。

### H7 場景直入
- 公式：〔精確時間／地點〕＋〔正在發生的事〕，第一句就站在現場
- 樣本：「那天夜診最後一號，晚上九點多⋯」（47）、吃完年夜飯姪女問「我可以去樓下玩了嗎？」（12）
- 適配：T4 故事、T9 暖心
- 注意：時間地點愈具體愈可信；模糊場景（「有一天」）無效。

### H8 目擊開場
- 公式：看到／遇到〔一件具體事〕＋我的反應
- 樣本：「看到了一種新型的減肥方法，就是把自己當成狗養。」（16）、「看到300壯士收攤，幾個心得：」（56）
- 適配：T7、T8 時事、T12
- 注意：目擊物本身要有戲，開場只是遞麥克風。

### H9 勞動量數字
- 公式：我花了〔時間〕／整理了〔大數字〕／分析了〔樣本數〕
- 樣本：「我花了兩天，整理了 150 招⋯」（50）、「我分析 202 篇貼文，發現真正有用的只有 6 篇」（52）
- 適配：T3 攻略、T6 實測覆盤
- 注意：數字必須是真的；勞動量鉤子觸發的是收藏，結尾記得接 E7。

### H10 證據視覺先行
- 公式：截圖／照片／影片就是第一行，文字只補一刀
- 樣本：離職單「看世界盃」（23）、監視器追出奧客（22）、兩品牌對照圖（73）
- 適配：S2 結構、T8、T9
- 注意：證據自帶可信度，但涉他人時想清楚法律與隱私（見 SKILL.md 鐵律 3）。

### H11 時機鉤
- 公式：在〔全台共時時刻〕發〔此刻才成立的一句話〕，發文時間本身就是內容
- 樣本：「今天台灣人集體失眠？」清晨 5 點發（15）、颱風直播中的鮭魚肚（05）
- 適配：T8 時事
- 注意：速度大於完美；過了那個小時就別發了。

### H12 嗆點
- 公式：〔先禮後兵〕＋一句立場（「很好，也恭喜～但關台灣屁事？」）
- 樣本：黃暐瀚（42）
- 適配：T1 爭議、V2 嗆辣
- 注意：對事不對人；波浪號與「恭喜」類緩衝詞控溫，避免純攻擊。

### H13 受害直陳
- 公式：我〔照規矩做了 X〕，結果〔被 Y〕＋求公評
- 樣本：「蠻好奇大家如果想離職會提早多久講 我的話都是依照勞基法⋯」（24）、面試趣被提告（63）、空服員同事發聲（69）
- 適配：T13 受害申訴、V10 義憤
- 注意：憤怒文傳播最快也最危險；單方敘事有翻車與法律風險，事實查核紀律最高（承 humanize L0）。

## 選用速查

| 你手上有什麼 | 用哪個鉤 |
| :-- | :-- |
| 一個很強的結論 | H1、H12 |
| 一個反常識的專業判斷 | H2 |
| 一張有戲的圖／截圖 | H10 |
| 一段真實經歷 | H7、H13 |
| 一大包整理好的乾貨 | H9＋H3 |
| 一個大家都能答的問題 | H5、H6 |
| 正在發生的全民事件 | H11、H8 |
| 什麼都沒有 | 先去收集素材，不要用套話硬寫（SKILL.md 鐵律 1） |
