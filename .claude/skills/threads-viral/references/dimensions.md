# 七維度規則卡（組合器值域）

每軸列出可選值。每個值是一張規則卡：怎麼寫、對應樣本（corpus.md 編號）、風險。
組合語法：`人設=P2 題材=T2 鉤子=H1 結構=S1 語氣=V1 節奏=R1 結尾=E1`，中文名或代號皆可。
未指定的軸依 recipes.md 高勝率配方自動選填；衝突依本檔末節互斥表裁決，使用者顯式指定的軸優先。

## D3 鉤子（第一行怎麼停住拇指）

值域與公式詳見 hooks.md（H1 到 H13）。硬規則：**摺疊線之前（前兩行）必須自帶完整張力**，不能靠「請展開看完」。

## D4 結構（骨架）

| 代號 | 值 | 規則卡 | 樣本 |

| S1 | 單句文 | 3 字到 2 行。台灣脆最強勢結構，字愈少愈可朝聖。發完不要補充，讓留言接。 | 01、02、34、15 |
| S2 | 視覺證據＋一句 | 截圖／照片／影片是主體，文案只補刀一句。證據自帶可信度（辭呈、監視器）。 | 23、05、30 |

## D6 節奏（格式紀律）

四種節奏擇一，以下硬規則全域適用：

| 代號 | 值 | 規則 |
| :-- | :-- | :-- |
| R1 | 極簡 | 單句、可無標點、可空格斷句（「請這位粉絲取關我一下 我只想默默當個i人」）。 |
| R2 | 標準脆節奏 | 一行一句，換行即標點，每行 5 到 20 字。 |
| R3 | 故事節奏 | 句句短但連貫成段，2 到 3 行一空行。 |
| R4 | 清單節奏 | 一行一項，項與項零廢話。 |

全域硬規則：

- 單則 ≤500 字元（中文 1 字＝1 字元）；裝不下改蓋樓，不硬塞。
- emoji 0 到 3 個，當視覺錨點不當裝飾雨；辦公正式內容 0。
- 驚嘆號全篇 ≤3，禁連續（！！）。
- 五行以上不換氣＝違規。
- 口語詞白名單：欸、啦、吧、靠、真的、超、有夠、笑死、好扯；中國用語零容忍（承 humanize localization.md）。
- 內文禁破折號（承 humanize F2），停頓用換行。
- 禁 markdown 符號（`**`、`#`、`-`），脆不渲染。
- hashtag：0 或 1 個主題標籤，同標籤別天天重複。

## 全域反模式

詳見 corpus.md「爆文絕不做的事」15 條。組合器層級的三條鐵律：**第一行不鋪陳、正文不放連結、結論不說教**（判斷權留給留言區）。
